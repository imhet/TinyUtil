# TinyUtil

Collect common utils on android.


## Usage

Step 1. Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}

Step 2. Add the dependency

	dependencies {
		compile 'com.github.imhet:TinyUtil:v0.9.2'
	}
